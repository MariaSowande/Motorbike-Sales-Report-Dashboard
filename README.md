# Motorbike-Sales-Report-Dashboard
There has been a good performance with sales over the past month but based on the budget for ads for the next 3 month, it is needed to niche down to the percentage of our high performing audience. For this reason, the raw sales dataset has to be analyzed before appropriate decision can be made.

## Dateset Used
- <a href="https://github.com/MariaSowande/Motorbike-Sales-Report-Dashboard/blob/main/Motorbike%20Dataset.xlsx">Dataset</a>

## Questions (KPIs)
- What is the average income of the buyers?
- Does the location of these individual have an effect on sales? 
- Which gender populates our sales? 
- Who should we target our ads to for the next quarter?

- Dashboard Interaction <a href="https://github.com/MariaSowande/Motorbike-Sales-Report-Dashboard/blob/main/Bike%20Sales%20Report%20DB.png">View Dashboard</a>

## Process
- Verified data for any missing variables
- Removed duplicates
- Formatted abbreviation to ensure unique representation
- Cleaned the data
- Created Pivot Table to display an appropriate solution to the questions
- Merged the Pivot Tables into a Dashboard fot better representation

  ## Conclusion
  From the report obtained, it can be concluded that, the middle-aged age group which ranges from 31 - 54 years buy more of the product, for this reason, higher percentage of the ad budget should be targeted towards them as this will have a greater effect on the business sales.
